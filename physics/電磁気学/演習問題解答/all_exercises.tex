\section*{過去問の傾向と解いておくとよい演習問題}
\addcontentsline{toc}{section}{過去問の傾向と解いておくとよい演習問題}

\subsection*{過去問の傾向（2023年度・2024年度・2024年度再試験）}

名古屋大学理学部の電磁気学過去問では、以下のテーマが繰り返し出題されています。

\begin{enumerate}
\item \textbf{物質中のMaxwell方程式}（毎年・全年度で出題）\\
静電磁場の4式を真電荷・分極電荷・真電流・磁化電流で書き、$\mathbf{D}$・$\mathbf{B}/\mu$ を用いた形に書き直す。時間変動を含むファラデー・アンペールの式。
\item \textbf{誘電体・導体の電場}（2023：誘電体板・導体板、2024・再：強誘電体球）\\
境界条件（$\mathbf{D}$ の法線成分、$\mathbf{E}$ の接線成分）、分極電荷、脱分極電場 $\mathbf{E}_p = -\mathbf{P}/(3\varepsilon_0)$、一様帯電球内部の電場。
\item \textbf{金属と電磁波}（2023：金属表面の反射・透過・減衰、2024再：伝導率 $\sigma(\omega)$）\\
反射率、金属内部での強度減衰、複素伝導率の実部・虚部、オームの法則の導出。
\item \textbf{分極の周波数応答}（2024・再：極性分子の $\chi_e(\omega)$）\\
緩和型の微分方程式から電気感受率を求める。$\mathrm{Re}[\chi_e]$ の図示、誘電率の虚部と電磁波の名前（マイクロ波・赤外線）。
\end{enumerate}

\subsection*{解いておくとよい演習問題（優先度順）}

過去問の類題となる演習問題を、出題傾向に合わせてまとめます。\textbf{特に第2回・第4回・第7回}は過去問と直結するため、優先して解いておくことを推奨します。

\paragraph{最優先（過去問と同一またはほぼ同一の内容）}
\begin{itemize}
\item \textbf{第2回 問題1}（Maxwellの方程式とエネルギー）\\
\quad $\to$ 過去問「物質中のMaxwell方程式」全年度問題1の類題。変位電流・ファラデー・アンペールの式に慣れる。
\item \textbf{第4回 問題1}（誘電体の内部電場）\\
\quad $\to$ 空洞電場 $-\mathbf{P}/(3\varepsilon_0)$、クラウジウス・モソッティ。2024・再の強誘電体球の(2-1)(2-2)の基礎。
\item \textbf{第4回 問題2}（外部電場中の誘電体球）\\
\quad $\to$ 2024・再「強誘電体球」の類題。外部電場中の誘電体球と、一様分極球の扱いが対応する。
\item \textbf{第7回 問題1}（液体中の有極性分子の分極と電気感受率）\\
\quad $\to$ 2024・再「極性分子の分極と $\chi_e(\omega)$」と同一テーマ。$\chi_e(\omega) = \chi_0/(1-\mathrm{i}\omega\tau)$ の導出と図示。
\item \textbf{第7回 問題4}（媒質境界面での電磁波の反射）\\
\quad $\to$ 2023年度問題3「金属表面における平面電磁波の反射・透過」の類題。
\item \textbf{第2回 問題2}（導体内の自由電子の運動とOhmの法則）\\
\quad $\to$ 2024再試験問題4「金属内の電子の運動と電気伝導率」の類題。$\mathbf{i}=nq\mathbf{v}$、$\sigma = nq^2/(\gamma m)$。
\item \textbf{第2回 問題3}（Ohmの法則と導体内の電荷の時間的減少）\\
\quad $\to$ 2024再試験問題4の補足。導体内の電荷の時間的減少と伝導の関係。
\end{itemize}

\paragraph{優先（誘電体・導体の境界、ガウスの法則）}
\begin{itemize}
\item \textbf{第4回 問題3}（誘電体境界面での鏡像法）、\textbf{第4回 問題4}（一様電場中の誘電体球）\\
\quad $\to$ 2023年度問題2「誘電体板・導体板の電場」の境界条件・斜め配置の考え方に通じる。
\item \textbf{第2回 問題4}（導体板に誘起される電荷）、\textbf{第2回 問題6}（接地された導体平面板と点電荷）\\
\quad $\to$ 2023年度問題2(2-4)「導体板の表面電荷」の類題。
\item \textbf{第1回 問題1}（球殻内外の電場と電位）\\
\quad $\to$ 一様帯電球の電場の計算（ガウスの法則）は、2024・再の強誘電体球(2-1)でそのまま使う。
\end{itemize}

\paragraph{余裕があれば（関連する考え方の確認）}
\begin{itemize}
\item 第1回 問題2（ポアソン方程式）、問題4（ビオ・サバールの法則の応用）\\
\item 第3回 問題2（誘電体入り同心導体球殻コンデンサー）、問題3（鏡像電荷）\\
\item 第5回・第6回は磁気、第7回 問題2（遅延ポテンシャル）・問題3（散乱）は発展的ながら参考になる。
\end{itemize}
