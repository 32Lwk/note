## 第1回 (exercise1.tex)

### 問題2: ポアソン方程式の解

1. **【誤り】領域1の「誤った計算」の扱い**  
   283–301行目で、最初の計算が負になり「符号を再確認する必要がある」と書いたあと、「正しい計算」に切り替えているが、**なぜ最初の計算が誤りか**（θ' 積分で $|r'-r| - |r'+r|$ の符号を $r<r'$ のときどう取るか）を一文で明示していない。初学者向けに「$r < r'$ のとき $|r'-r| = r'-r$、$|r'+r| = r'+r$ なので……」と短く補足するとよい。

2. **【誤り】領域2の電位の式変形**  
   354行目で  
   $\frac{1}{r}\cdot\frac{r^3-b^3}{3} = \frac{r^2 - b^3/r}{3}$  
   としているが、正しくは  
   $\frac{r^3-b^3}{3r} = \frac{r^2}{3} - \frac{b^3}{3r}$  
   である。  
   $\frac{r^2}{3} - \frac{b^3}{3r}$ と $\frac{a^2-r^2}{2}$ をまとめると、最終形  
   $\frac{\rho}{6\varepsilon_0}[3a^2 - r^2 - \frac{2b^3}{r}]$  
   と一致する。**式中の $\frac{r^2 - b^3/r}{3}$ を $\frac{r^2}{3} - \frac{b^3}{3r}$ に修正**する必要がある。

### 問題4: ビオ・サバールの法則の応用 (4-2)

3. **【不備】アンペールの法則の適用**  
   642行目「ループ内を貫く電流は $n I \times 2\pi r$」は誤り。無限ソレノイドでは、軸に平行な長さ $L$ のループを考え、貫く電流は **$n L I$**（単位長さあたり $n$ 回巻き、1巻きあたり $I$）。  
   正しくは「長さ $L$ の区間で $B L = \mu_0 n L I$ より $B = \mu_0 n I$」と書くべき。**$2\pi r$ は使わず、軸方向の長さ $L$ で説明**するとよい。

### 問題6: 電磁波の波動方程式 (6-3)

4. **【説明不足】導体中の $k$ の実部・虚部**  
   $k^2 = \varepsilon_0\mu_0\omega^2 + i\mu_0\sigma\omega$ から $k_r$, $k_i$ の連立まで書いているが、**$k_r$, $k_i$ の陽な式**（または $\sigma$ が小さいときの近似）まで書いていない。減衰長 $1/k_i$ の議論のため、少なくとも「良導体近似」など一言あるとよい。

## 第2回 (exercise2.tex)

### 問題6: 接地された導体平面板と点電荷

5. **【誤り】電荷面密度と総電荷の符号**  
   - 導体表面（$z=0$）直上での電場の $z$ 成分は、$q$ と鏡像 $-q$ の両方の寄与で **$E_z = -qd/(2\pi\varepsilon_0 R^3)$**（負）である。  
   - 法線を真空側（$+z$）に取ると $\sigma = \varepsilon_0 E_z$ なので、**$\sigma$ は負**で  
     $\sigma(x,y) = -\frac{qd}{2\pi[x^2+y^2+d^2]^{3/2}}$  
     が正しい。  
   - 現在の解答は $\sigma$ を正で書いているため、積分すると $Q = q$ になるが、接地導体の誘起電荷の総量は **$Q = -q$** である。  
   **修正**: $E_z$ の計算で鏡像電荷 $-q$ の寄与の符号を正し、$\sigma = -\frac{qd}{2\pi[x^2+y^2+d^2]^{3/2}}$ とし、積分で $Q = -q$ が導かれるようにする。

### 問題1: 外部磁場中の常磁性体球 (1-1)

9. **【説明不足】磁化電流の積分計算**  
   ビオ・サバールによる積分を進めると $\int_0^\pi \sin^3\theta\cos\theta\,d\theta = 0$ となり、そのままでは答えが出ない。ここで「より正確には」と**既知の公式** $\boldsymbol{B}_M = \frac{2\mu_0\boldsymbol{M}}{3}$ に飛んでいる。  
   - **なぜ直接積分が 0 になるか**（対称性・符号の取り方の注意）を1文で書く。  
   - あるいは「磁化球内部の磁場は磁位から求まる」など、**公式の根拠**を一言書くとよい。
