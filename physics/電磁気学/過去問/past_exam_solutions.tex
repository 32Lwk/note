\section*{この解答解説について（初学者の方へ）}
\addcontentsline{toc}{section}{この解答解説について（初学者の方へ）}

各問題の解答では、次のような構成を心がけています。

\begin{itemize}
\item \textbf{この問題のポイント}：その問題で何を理解することが大切か、どのような流れで解くかを短くまとめています。
\item \textbf{解き方の流れ}：各小問をどの順に、どの公式や法則を使って解くかを箇条書きにしています。
\item \textbf{用語の説明}：分極、磁化、電束密度、誘電率、透磁率など、問題で使う用語を初出付近で説明しています。
\item \textbf{使用する物理法則}：Maxwell方程式、境界条件、オームの法則など、どの法則に基づいて式を立てているかを明示しています。
\item \textbf{計算のステップ}：式を立てるときに「ステップ1」「ステップ2」と区切り、途中を省略せずに記述しています。
\item \textbf{なぜこのように求まるか}：答えの式が成り立つ理由や、物理的な意味を補足しています。
\item \textbf{図}：設定の概念図、電場・磁場の向き、グラフなどを入れています。
\end{itemize}

\noindent
数式の変形では「明らかに」「同様に」に頼らず、途中のステップを書いています。ベクトル計算では成分を明示し、積分では変数変換や部分積分の過程を示しています。

\paragraph{読むときのコツ（初学者の方へ）}
\begin{itemize}
\item 各小問の最終的な数式の答えは「【解答】」の直後で四角（\boxed）で囲んであります。途中の式には番号を付けず、参照する式にのみ数式番号を付けています。
\item まず各問題の「この問題のポイント」と「解き方の流れ」を読むと、何を求めるか・どの順で式を立てるかが把握しやすくなります。
\item 用語（分極、磁化、電束密度、境界条件など）は初出付近の「用語の説明」で定義しています。わからなくなったらそこに戻って確認してください。
\item 「なぜ～か」「原理的な説明」「物理的考察」の段落は、式の意味や物理的なイメージを理解するためのものです。答えの式だけではなく、なぜその式になるかも読むと理解が深まります。
\item 図は設定のイメージや式の対応関係を示しています。図のキャプションにも式や条件を書いているので、図と本文を対応させて読むとよいです。
\item 類題一覧では、各過去問が他年度のどの問題と対応するか、および演習問題解説（\texttt{all\_exercises.pdf}）のどの回・問題が類題かをまとめています。目次および各問題の見出しにも「類題：○○」を併記しているので、復習の際に参照してください。
\item 2024年度・再試験では「2023年度問題1と同一」のように他年度を参照することがあります。その場合は該当年度の該当問題を開き、「解き方の流れ」と「なぜ～か」の段落をあわせて読むと理解しやすくなります。
\end{itemize}

\section*{類題一覧}
\addcontentsline{toc}{section}{類題一覧}

以下の表に、同じまたは類似した問題が他年度のどの問題番号で出題されているか、および電磁気学 演習問題解答（\texttt{all\_exercises.pdf}）のどの回・どの問題が類題かをまとめます。目次では各問題の見出しに「類題：○○」も併記しています。

\begin{itemize}
\item \textbf{2023年度 期末試験（2月7日）}\\
　問題1：物質中のMaxwell方程式（静電磁場の4式、$\mathbf{D}$・$\mathbf{B}/\mu$ を用いた書き換え、時間変動を含む2式）。類題：2024年度 問題1、2024年度再試験 問題1。演習問題解説：第2回 問題1（Maxwellの方程式とエネルギー、変位電流・ファラデー・アンペールの式）。\\
　問題2：誘電体板・導体板の電場（垂直・斜め配置、内部の $\mathbf{D}$・$\mathbf{E}$、分極電荷の面密度、導体表面電荷）。類題：なし（2024・2024再は強誘電体球）。演習問題解説：第4回 問題2（外部電場中の誘電体球）、問題3（誘電体境界面での鏡像法）、問題4（一様電場中の誘電体球）。第2回 問題4（導体板に誘起される電荷）、問題6（接地された導体平面板と点電荷）。\\
　問題3：金属表面における平面電磁波の反射・透過（反射率 $R$、金属内部での強度減衰、厚み $d$ の金属板透過）。類題：なし（2024・2024再は極性分子・伝導率）。演習問題解説：第7回 問題4（媒質境界面での電磁波の反射）。
\item \textbf{2024年度 期末試験（2月4日）}\\
　問題1：物質中のMaxwell方程式。類題：2023年度 問題1、2024年度再試験 問題1。演習問題解説：第2回 問題1。\\
　問題2：均一に分極した強誘電体球（一様帯電球内部の電場、$\mathbf{E}_p = -\mathbf{P}/(3\varepsilon_0)$、電気力線・電束線、$\mathbf{D}$ と $\varepsilon_0\mathbf{E}$ の関係）。類題：2024年度再試験 問題2（(2-1)(2-2)のみ）。演習問題解説：第4回 問題1（誘電体の内部電場、空洞電場 $-\mathbf{P}/(3\varepsilon_0)$・クラウジウス・モソッティ）、問題2（外部電場中の誘電体球）。第1回 問題1（球殻・ガウスの法則）は一様帯電球の電場の計算に類似。\\
　問題3：液体中の有極性分子の分極ダイナミクス（電気感受率 $\chi_e(\omega)$、$\mathrm{Re}[\chi_e]$ の図示、誘電率の虚部 $\varepsilon''$ と電磁波の名前）。類題：2024年度再試験 問題3（(3-1)(3-2)のみ）。演習問題解説：第7回 問題1（液体中の有極性分子の分極と電気感受率）。
\item \textbf{2024年度 再試験（2月6日）}\\
　問題1：物質中のMaxwell方程式。類題：2023年度 問題1、2024年度 問題1。演習問題解説：第2回 問題1。\\
　問題2：強誘電体球の電場（(2-1)(2-2)のみ：一様帯電球内部の電場、$\mathbf{E}_p = -\mathbf{P}/(3\varepsilon_0)$）。類題：2024年度 問題2。演習問題解説：第4回 問題1、問題2。第1回 問題1。\\
　問題3：液体中の極性分子の分極と電気感受率（$\chi_e(\omega)$、実部の図示）。類題：2024年度 問題3。演習問題解説：第7回 問題1。\\
　問題4：金属内の電子の運動と電気伝導率（電流密度と速度、オームの法則 $\sigma$、複素伝導率 $\sigma(\omega)$ の実部・虚部）。類題：なし（他年度では出題なし）。演習問題解説：第2回 問題2（導体内の自由電子の運動とOhmの法則）、問題3（Ohmの法則と導体内の電荷の時間的減少）。
\end{itemize}
