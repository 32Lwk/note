# 問題 5-2 解答: 双子のパラドックス

## 問題
慣性系S'において、宇宙船が原点にあり、慣性系Sに対してx方向に速度vで運動している。t = t' = 0のとき、S系とS'系の原点OとO'は一致する。宇宙船はある点Pまで進み、そこで外力を受けて瞬時に速度を-vに変え、S系の原点Oに戻る。光速をc、β = v/cとする。

## 解答

### (i) 点P到達時の時刻の関係

**問題:** 宇宙船が点Pに到達したとき、S系での時刻がt₁、S'系での時刻がt₂のとき、t₂をt₁で表せ。

**解答:**

ローレンツ変換を用いる。S'系はS系に対してx方向に速度vで運動している。

S'系からS系への変換:
$$t = \gamma\left(t' + \frac{vx'}{c^2}\right)$$

点Pは宇宙船と共に運動しているので、S'系では静止している。したがって、S'系での時刻$t_2$とS系での時刻$t_1$の関係は、時間の遅れにより:

$$t_2 = t_1\sqrt{1-\beta^2}$$

**答え:** $t_2 = t_1\sqrt{1-\beta^2}$

---

### (ii) 宇宙船から見たS系原点の時刻

**問題:** 宇宙船の視点から、点Pに到達したとき、S系の原点での時刻t₃をt₂で表せ。

**解答:**

宇宙船（S'系）から見ると、S系は-x方向に速度vで運動している。

点P到達時（S'系での時刻$t_2$、S'系での座標$x'_P$）に、S系の原点O（S系での座標$x = 0$）の時刻を求める。

S'系からS系への変換:
$$t = \gamma\left(t' + \frac{vx'}{c^2}\right)$$

S系の原点OのS'系での座標を求める。S'系から見ると、S系の原点は-x方向に速度vで運動している。

時刻$t' = 0$のとき、$x' = 0$（原点が一致）
時刻$t' = t_2$のとき、S系の原点のS'系での座標は:
$$x'_O = -v t_2$$

したがって、S系の原点での時刻:
$$t_3 = \gamma\left(t_2 + \frac{v(-v t_2)}{c^2}\right) = \gamma\left(t_2 - \frac{v^2 t_2}{c^2}\right) = \gamma t_2(1 - \beta^2) = t_2\sqrt{1-\beta^2}$$

**答え:** $t_3 = t_2\sqrt{1-\beta^2}$

---

### (iii) 時空図

**問題:** 宇宙船の軌跡を時空図に描け。S系の時空軸を直交として。

**解答:**

S系の時空図（ct軸とx軸が直交）:

1. **往路（0 ≤ t ≤ t₁）:**
   - 宇宙船は速度vで+x方向に運動
   - 軌跡: $x = v t$（直線、傾き$1/v$）

2. **点Pでの速度変化（t = t₁）:**
   - 瞬時に速度が-vに変化

3. **復路（t₁ ≤ t ≤ t₂）:**
   - 宇宙船は速度-vで-x方向に運動
   - 軌跡: $x = v t_1 - v(t - t_1) = 2v t_1 - v t$（直線、傾き$-1/v$）

時空図では:
- 往路: $(ct, x) = (c\tau, v\tau)$ for $0 \leq \tau \leq t_1$
- 復路: $(ct, x) = (c(2t_1 - \tau), v(2t_1 - \tau))$ for $t_1 \leq \tau \leq 2t_1$

![時空図](problem5-2_spacetime_diagram.png)

上図はS系での時空図である。赤い実線が宇宙船の軌跡を示し、往路と復路が点Pで折れ曲がっている。点線は光の軌跡（45度の線）とS'系の座標軸（往路時）を示している。

### (iv) 速度変化時の時刻の進み

**問題:** 時空図を用いて、点Pで速度がvから-vに変化する瞬間、宇宙船から見たS系の時刻が$2(t_1 - t_3)$だけ進んでいることを説明せよ。

**解答:**

(i)と(ii)の結果から:
- $t_1 = \frac{t_2}{\sqrt{1-\beta^2}}$
- $t_3 = t_2\sqrt{1-\beta^2}$

したがって:
$$t_1 - t_3 = \frac{t_2}{\sqrt{1-\beta^2}} - t_2\sqrt{1-\beta^2} = t_2\left(\frac{1}{\sqrt{1-\beta^2}} - \sqrt{1-\beta^2}\right) = t_2 \cdot \frac{1 - (1-\beta^2)}{\sqrt{1-\beta^2}} = \frac{t_2\beta^2}{\sqrt{1-\beta^2}}$$

速度変化の瞬間、宇宙船の慣性系がS'系から別の慣性系（S''系、速度-v）に変わる。

速度変化前（S'系）: S系の原点の時刻は$t_3 = t_2\sqrt{1-\beta^2}$

速度変化後（S''系）: 新しい慣性系から見ると、同時刻の定義が変わる。S''系はS系に対して-x方向に速度vで運動している。

S''系からS系への変換:
$$t = \gamma\left(t'' + \frac{(-v)x''}{c^2}\right) = \gamma\left(t'' - \frac{vx''}{c^2}\right)$$

点Pでの速度変化時、S''系での時刻を$t''_P$とすると、S系の原点（$x = 0$）のS''系での座標は、速度変化前と同様に$-v t''_P$（ただし、S''系から見るとS系は+x方向に運動）。

より正確には、速度変化の瞬間、S系の原点の時刻は、新しい同時刻の定義により$t_1$になる。

時刻の進み:
$$\Delta t = t_1 - t_3 = \frac{t_2}{\sqrt{1-\beta^2}} - t_2\sqrt{1-\beta^2} = t_2\left(\frac{1}{\sqrt{1-\beta^2}} - \sqrt{1-\beta^2}\right)$$

往路と復路で対称なので、合計で$2(t_1 - t_3)$だけ進む。

![速度変化時の同時刻の相対性](problem5-2_part4_diagram.png)

上図は、点Pでの速度変化時の同時刻の相対性を示している。

- **青い破線**: S'系（速度変化前）の同時刻線。点Pでの時刻$t_2$と同時刻の点を結ぶ線である。この線とS系の原点（$x = 0$）の交点が、S'系から見たS系原点の時刻$t_3$を示す（青い四角）。
- **緑の破線**: S''系（速度変化後）の同時刻線。点Pでの時刻と同時刻の点を結ぶ線である。この線とS系の原点の交点が、S''系から見たS系原点の時刻$t_1$を示す（緑の四角）。
- **紫色の矢印**: 時刻の進み$\Delta t = t_1 - t_3$を視覚的に示している。

速度変化により、宇宙船の慣性系がS'系からS''系に変わることで、同時刻の定義が変わる。その結果、S系の原点での時刻が$t_3$から$t_1$に「飛ぶ」ように見える。この時刻の差は$t_1 - t_3$である。

往路と復路で対称的な状況が起こるため、合計で$2(t_1 - t_3)$だけ時刻が進むように見える。

**答え:** 速度変化により同時刻の定義が変わり、S系の原点の時刻が$2(t_1 - t_3)$だけ進む。

### (v) 再会時の時計の遅れ

**問題:** 再会したとき、どちらの時計が遅れているか。

**解答:**

宇宙船の往復時間:
- S系から見た往復時間: $T = 2t_1$（往路$t_1$、復路$t_1$）
- 宇宙船内の時間: 往路で$t_2 = t_1\sqrt{1-\beta^2}$、復路も対称なので$t_2 = t_1\sqrt{1-\beta^2}$
- 合計: $T' = 2t_2 = 2t_1\sqrt{1-\beta^2} = T\sqrt{1-\beta^2}$

したがって、宇宙船内の時計は$T' = T\sqrt{1-\beta^2} < T$なので、**宇宙船内の時計が遅れている**。

これは、宇宙船が加速（速度変化）を経験するため、慣性系ではないからである。S系は慣性系のままなので、S系の時計が正しい時間を刻む。

**答え:** 宇宙船内の時計が遅れている。
