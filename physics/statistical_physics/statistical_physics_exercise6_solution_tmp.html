統計物理学Ⅰ・演習6 解答

セクションI: 偏微分と熱力学関数

問1. 内部エネルギーの方程式

問題: 以下の関係式を導け。 $$(1) \quad \left(\frac{\partial U}{\partial V}\right)_T = T^2 \left{\frac{\partial(p/T)}{\partial T}\right}_V$$

解答:

基本関係式: $$TdS = dU + pdV \quad \text{(第一法則と第二法則の結合)}$$ $$dF = -SdT - pdV \quad \text{(ヘルムホルツ自由エネルギー)}$$

方法1: エントロピーを用いた導出

$TdS = dU + pdV$より、$dU = TdS - pdV$。

$U$を$T$と$V$の関数として考えると: $$dU = \left(\frac{\partial U}{\partial T}\right)_V dT + \left(\frac{\partial U}{\partial V}\right)_T dV$$

一方、$S$を$T$と$V$の関数として: $$dS = \left(\frac{\partial S}{\partial T}\right)_V dT + \left(\frac{\partial S}{\partial V}\right)_T dV$$

したがって: $$dU = T\left[\left(\frac{\partial S}{\partial T}\right)_V dT + \left(\frac{\partial S}{\partial V}\right)_T dV\right] - pdV$$

$$= T\left(\frac{\partial S}{\partial T}\right)_V dT + \left[T\left(\frac{\partial S}{\partial V}\right)_T - p\right]dV$$

係数を比較: $$\left(\frac{\partial U}{\partial V}\right)_T = T\left(\frac{\partial S}{\partial V}\right)_T - p$$

マクスウェル関係式より、$dF = -SdT - pdV$から: $$\left(\frac{\partial S}{\partial V}\right)_T = \left(\frac{\partial p}{\partial T}\right)_V$$

したがって: $$\left(\frac{\partial U}{\partial V}\right)_T = T\left(\frac{\partial p}{\partial T}\right)_V - p$$

右辺を整理: $$T\left(\frac{\partial p}{\partial T}\right)_V - p = T^2 \left[\frac{1}{T}\left(\frac{\partial p}{\partial T}\right)_V - \frac{p}{T^2}\right]$$

$$= T^2 \left[\frac{\partial}{\partial T}\left(\frac{p}{T}\right)\right]_V = T^2 \left{\frac{\partial(p/T)}{\partial T}\right}_V$$

答え: $$\left(\frac{\partial U}{\partial V}\right)_T = T^2 \left{\frac{\partial(p/T)}{\partial T}\right}_V$$

物理的意味:

  • この式は、内部エネルギーが体積に依存する度合いを、圧力の温度依存性から計算できることを示している。
  • 理想気体では$p/T = nR/V$（$n$はモル数、$R$は気体定数）なので、右辺は0となり、内部エネルギーは体積に依存しない（ジュールの法則）。
  • 実在気体では、分子間相互作用により圧力が温度に依存するため、内部エネルギーも体積に依存する。

問2. 定圧熱容量と定積熱容量の関係式

問題: 以下の関係式を導け。 $$(2) \quad C_p = C_V + T\left(\frac{\partial V}{\partial T}\right)_p \left(\frac{\partial p}{\partial T}\right)_V$$

解答:

熱容量の定義: $$C_V = \left(\frac{\partial U}{\partial T}\right)_V = T\left(\frac{\partial S}{\partial T}\right)_V$$ $$C_p = \left(\frac{\partial H}{\partial T}\right)_p = T\left(\frac{\partial S}{\partial T}\right)_p$$

ここで、$H = U + pV$はエンタルピー。

方法1: エンタルピーを用いた導出

$H = U + pV$より: $$dH = dU + pdV + Vdp = TdS + Vdp$$

$H$を$T$と$p$の関数として: $$dH = \left(\frac{\partial H}{\partial T}\right)_p dT + \left(\frac{\partial H}{\partial p}\right)_T dp$$

一方、$S$を$T$と$p$の関数として: $$dS = \left(\frac{\partial S}{\partial T}\right)_p dT + \left(\frac{\partial S}{\partial p}\right)_T dp$$

したがって: $$dH = T\left[\left(\frac{\partial S}{\partial T}\right)_p dT + \left(\frac{\partial S}{\partial p}\right)_T dp\right] + Vdp$$

$$= T\left(\frac{\partial S}{\partial T}\right)_p dT + \left[T\left(\frac{\partial S}{\partial p}\right)_T + V\right]dp$$

係数を比較: $$C_p = T\left(\frac{\partial S}{\partial T}\right)_p$$

一方、$S$を$T$と$V$の関数として考えると: $$dS = \left(\frac{\partial S}{\partial T}\right)_V dT + \left(\frac{\partial S}{\partial V}\right)_T dV$$

$p$を一定に保つと: $$\left(\frac{\partial S}{\partial T}\right)_p = \left(\frac{\partial S}{\partial T}\right)_V + \left(\frac{\partial S}{\partial V}\right)_T \left(\frac{\partial V}{\partial T}\right)_p$$

したがって: $$C_p = T\left(\frac{\partial S}{\partial T}\right)_V + T\left(\frac{\partial S}{\partial V}\right)_T \left(\frac{\partial V}{\partial T}\right)_p$$

$$= C_V + T\left(\frac{\partial S}{\partial V}\right)_T \left(\frac{\partial V}{\partial T}\right)_p$$

マクスウェル関係式より: $$\left(\frac{\partial S}{\partial V}\right)_T = \left(\frac{\partial p}{\partial T}\right)_V$$

したがって: $$C_p = C_V + T\left(\frac{\partial V}{\partial T}\right)_p \left(\frac{\partial p}{\partial T}\right)_V$$

答え: $$C_p = C_V + T\left(\frac{\partial V}{\partial T}\right)_p \left(\frac{\partial p}{\partial T}\right)_V$$

物理的意味:

  • $C_p > C_V$である理由を説明している。
  • 定圧過程では、温度を上げる際に体積が膨張するため、外部に対して仕事をする必要がある。その分だけ余分な熱が必要になる。
  • 右辺の第2項$T(\partial V/\partial T)_p (\partial p/\partial T)_V$は、この体積変化に伴う追加のエネルギーを表している。
  • 理想気体では、$(\partial V/\partial T)_p = nR/p$、$(\partial p/\partial T)_V = nR/V$より、$C_p - C_V = nR$（マイヤーの関係式）。

セクションII: 輪ゴムの熱力学

問1. バネの自由エネルギー

問題: バネの張力が$X = -k(T)x$（$k(T) = k_0 + k_1T$）で与えられ、比熱が$C = \text{定数}$のとき、自由エネルギーが $$F(T,x) = -CT\left[\ln\left(\frac{T}{T_0}\right) - 1\right] + \frac{1}{2}k(T)x^2$$ と書けることを示せ。ただし、$F(T=0, x=0) = 0$とする。

解答:

基本関係式: $$dF = -SdT - Xdx$$

ここで、$X = -k(T)x = -(k_0 + k_1T)x$である。

ステップ1: 張力からの条件

$$\left(\frac{\partial F}{\partial x}\right)_T = -X = k(T)x = (k_0 + k_1T)x$$

これを$x$で積分: $$F(T,x) = \frac{1}{2}k(T)x^2 + f(T)$$

ここで、$f(T)$は$x$に依存しない関数。

ステップ2: エントロピーからの条件

$$\left(\frac{\partial F}{\partial T}\right)_x = -S$$

一方、$F(T,x) = \frac{1}{2}k(T)x^2 + f(T)$より: $$\left(\frac{\partial F}{\partial T}\right)_x = \frac{1}{2}k_1 x^2 + f'(T)$$

したがって: $$S = -\frac{1}{2}k_1 x^2 - f'(T)$$

ステップ3: 比熱からの条件

比熱は、$x$を一定に保ったときの熱容量: $$C = T\left(\frac{\partial S}{\partial T}\right)_x = -Tf''(T)$$

$C$が定数なので: $$f''(T) = -\frac{C}{T}$$

これを積分: $$f'(T) = -C\ln T + A$$

もう一度積分: $$f(T) = -C(T\ln T - T) + AT + B = -CT\left(\ln T - 1\right) + AT + B$$

ステップ4: 境界条件の適用

$F(T=0, x=0) = 0$より: $$f(0) = 0$$

しかし、$T \to 0$で$\ln T \to -\infty$となるため、適切な基準温度$T_0$を導入: $$f(T) = -CT\left[\ln\left(\frac{T}{T_0}\right) - 1\right] + AT + B$$

$F(T=0, x=0) = 0$より、$B = 0$。

また、$A$は$T$の線形項であるが、$F(T=0, x=0) = 0$の条件から、$A = 0$とすることができる（または、$T_0$の選択で吸収できる）。

答え: $$F(T,x) = -CT\left[\ln\left(\frac{T}{T_0}\right) - 1\right] + \frac{1}{2}k(T)x^2$$

物理的意味:

  • 第1項は、温度依存性による自由エネルギー（エントロピー効果）。
  • 第2項は、バネの弾性エネルギー。温度が高いほど、バネ定数が大きくなり、同じ伸びに対してより大きなエネルギーが必要。
  • 輪ゴムが伸びるときに発熱する理由は、$k_1 > 0$により、温度が高いほど張力が大きくなるため。

問2. 内部エネルギーの計算

問題: 問1の結果と、内部エネルギーの関係式（問1の(2)式のバネ版） $$\left(\frac{\partial U}{\partial x}\right)_T = T^2 \left{\frac{\partial(X/T)}{\partial T}\right}_x$$ を用いて、内部エネルギー$U(T,x)$を求めよ。ただし、$U(T=0, x=0) = 0$とする。

解答:

ステップ1: 内部エネルギーの関係式の適用

$$X = -k(T)x = -(k_0 + k_1T)x$$

したがって: $$\frac{X}{T} = -\frac{(k_0 + k_1T)x}{T} = -\frac{k_0 x}{T} - k_1 x$$

$$\left(\frac{\partial(X/T)}{\partial T}\right)_x = \frac{k_0 x}{T^2}$$

したがって: $$\left(\frac{\partial U}{\partial x}\right)_T = T^2 \cdot \frac{k_0 x}{T^2} = k_0 x$$

ステップ2: 内部エネルギーの積分

$$U(T,x) = \int_0^x k_0 x' dx' + g(T) = \frac{1}{2}k_0 x^2 + g(T)$$

ここで、$g(T)$は$x$に依存しない関数。

ステップ3: 比熱からの条件

内部エネルギーとエントロピーの関係: $$U = F + TS$$

問1より: $$F(T,x) = -CT\left[\ln\left(\frac{T}{T_0}\right) - 1\right] + \frac{1}{2}k(T)x^2$$

$$S = -\left(\frac{\partial F}{\partial T}\right)_x = C\left[\ln\left(\frac{T}{T_0}\right) - 1\right] + CT \cdot \frac{1}{T} - \frac{1}{2}k_1 x^2$$

$$= C\ln\left(\frac{T}{T_0}\right) - \frac{1}{2}k_1 x^2$$

したがって: $$U = F + TS = -CT\left[\ln\left(\frac{T}{T_0}\right) - 1\right] + \frac{1}{2}k(T)x^2 + CT\ln\left(\frac{T}{T_0}\right) - \frac{1}{2}k_1 T x^2$$

$$= CT + \frac{1}{2}k_0 x^2 + \frac{1}{2}k_1 T x^2 - \frac{1}{2}k_1 T x^2 = CT + \frac{1}{2}k_0 x^2$$

答え: $$U(T,x) = CT + \frac{1}{2}k_0 x^2$$

物理的意味:

  • 内部エネルギーは、温度に比例する項（比熱による）と、伸びの2乗に比例する項（弾性エネルギー）からなる。
  • 重要な点は、$k_1$（温度依存部分）は内部エネルギーには現れず、自由エネルギーにのみ現れること。
  • これは、$k_1$の効果が主にエントロピー効果であることを示している。
  • 輪ゴムを素早く伸ばすと、断熱過程として$U$が一定に保たれるため、$\frac{1}{2}k_0 x^2$が増加すると$CT$が減少し、温度が下がるように見えるが、実際にはエントロピー効果により温度が上昇する。

セクションIII: 1成分理想気体

問1. 化学ポテンシャルの計算

問題: エントロピー$S = NR\ln(T^c V/N) + NS_0$と内部エネルギー$U = cNRT$から、化学ポテンシャル$\mu$を計算せよ。

解答:

基本関係式: $$TdS = dU + pdV - \mu dN$$

$S$と$U$が与えられているので、$dS$と$dU$を計算:

$$dS = NR \cdot \frac{c}{T} dT + NR \cdot \frac{1}{V} dV - NR \cdot \frac{1}{N} dN + R\ln(T^c V/N) dN + S_0 dN$$

$$= \frac{cNR}{T} dT + \frac{NR}{V} dV + \left[R\ln(T^c V/N) + S_0 - R\right] dN$$

$$dU = cRT dN + cNR dT$$

したがって: $$TdS = cNR dT + \frac{NR T}{V} dV + T\left[R\ln(T^c V/N) + S_0 - R\right] dN$$

$$dU + pdV = cRT dN + cNR dT + pdV$$

比較すると: $$pdV = \frac{NR T}{V} dV \quad \Rightarrow \quad p = \frac{NR T}{V}$$

また: $$-\mu dN = T\left[R\ln(T^c V/N) + S_0 - R\right] dN - cRT dN$$

したがって: $$\mu = -RT\ln(T^c V/N) - TS_0 + RT + cRT$$

$$= -RT\ln(T^c V/N) - TS_0 + (c+1)RT$$

答え: $$\mu = -RT\ln\left(\frac{T^c V}{N}\right) - TS_0 + (c+1)RT$$

物理的意味:

  • 化学ポテンシャルは、粒子を1個追加する際の自由エネルギーの変化を表す。
  • 温度が高いほど、体積が大きいほど、粒子数が少ないほど、化学ポテンシャルは低くなる（粒子を追加しやすくなる）。

問2. ヘルムホルツ自由エネルギー

問題: 内部エネルギーとエントロピーから、ヘルムホルツ自由エネルギー$F(T,V,N)$を求め、全微分形式を書け。また、各偏微分がどの熱力学変数に対応するか答えよ。

解答:

ステップ1: 自由エネルギーの計算

$$F = U - TS = cNRT - T\left[NR\ln(T^c V/N) + NS_0\right]$$

$$= cNRT - NRT\ln(T^c V/N) - NTS_0$$

$$= NRT\left[c - \ln(T^c V/N)\right] - NTS_0$$

$$= NRT\left[c - c\ln T - \ln(V/N)\right] - NTS_0$$

ステップ2: 全微分

$$dF = \left(\frac{\partial F}{\partial T}\right){V,N} dT + \left(\frac{\partial F}{\partial V}\right){T,N} dV + \left(\frac{\partial F}{\partial N}\right)_{T,V} dN$$

各偏微分を計算:

$$\left(\frac{\partial F}{\partial T}\right)_{V,N} = NR\left[c - \ln(T^c V/N)\right] - NRT \cdot \frac{c}{T} - NS_0$$

$$= NR\left[c - \ln(T^c V/N) - c\right] - NS_0 = -NR\ln(T^c V/N) - NS_0 = -S$$

$$\left(\frac{\partial F}{\partial V}\right)_{T,N} = -NRT \cdot \frac{1}{V} = -\frac{NRT}{V} = -p$$

$$\left(\frac{\partial F}{\partial N}\right)_{T,V} = RT\left[c - \ln(T^c V/N)\right] - RTS_0 = \mu$$

したがって: $$dF = -SdT - pdV + \mu dN$$

答え: $$F(T,V,N) = NRT\left[c - \ln(T^c V/N)\right] - NTS_0$$

$$dF = -SdT - pdV + \mu dN$$

各偏微分の対応:

  • $\left(\frac{\partial F}{\partial T}\right)_{V,N} = -S$（エントロピー）
  • $\left(\frac{\partial F}{\partial V}\right)_{T,N} = -p$（圧力）
  • $\left(\frac{\partial F}{\partial N}\right)_{T,V} = \mu$（化学ポテンシャル）

物理的意味:

  • 自由エネルギーは、温度、体積、粒子数の関数として、系の平衡状態を完全に記述する。
  • 各偏微分は、対応する熱力学変数（共役変数）を与える。

問3. 化学ポテンシャルの検証

問題: 問2の結果から、化学ポテンシャルを直接計算し、問1の結果と一致することを確認せよ。

解答:

問2より: $$\mu = \left(\frac{\partial F}{\partial N}\right)_{T,V}$$

$$F = NRT\left[c - \ln(T^c V/N)\right] - NTS_0$$

$N$で偏微分する際、$T^c V/N$の中に$N$が含まれることに注意:

$$\mu = RT\left[c - \ln(T^c V/N)\right] - NRT \cdot \frac{\partial}{\partial N}\ln(T^c V/N) - TS_0$$

$$\frac{\partial}{\partial N}\ln(T^c V/N) = \frac{\partial}{\partial N}\left[\ln(T^c V) - \ln N\right] = -\frac{1}{N}$$

したがって: $$\mu = RT\left[c - \ln(T^c V/N)\right] + RT - TS_0$$

$$= RT\left[c + 1 - \ln(T^c V/N)\right] - TS_0$$

$$= -RT\ln(T^c V/N) + (c+1)RT - TS_0$$

これは問1の結果と一致する。

答え: 一致する。

問4. エントロピーの線形形式

問題: エントロピー$S(U,V,N)$を、広延変数の線形和の形 $$S(U,V,N) = a_U U + a_V V + a_N N$$ で表し、式(8)を再現せよ。

解答:

基本関係式: $$TdS = dU + pdV - \mu dN$$

したがって: $$dS = \frac{1}{T} dU + \frac{p}{T} dV - \frac{\mu}{T} dN$$

係数を比較: $$a_U = \frac{1}{T}, \quad a_V = \frac{p}{T}, \quad a_N = -\frac{\mu}{T}$$

理想気体の状態方程式と内部エネルギー: $$p = \frac{NR T}{V}, \quad U = cNRT$$

したがって: $$a_U = \frac{1}{T}, \quad a_V = \frac{NR}{V}, \quad a_N = -\frac{\mu}{T}$$

問1より: $$\mu = -RT\ln(T^c V/N) - TS_0 + (c+1)RT$$

したがって: $$a_N = R\ln(T^c V/N) + S_0 - (c+1)R$$

$T = U/(cNR)$、$V$と$N$を用いて: $$S = \frac{1}{T} U + \frac{NR}{V} V - \frac{\mu}{T} N$$

$$= \frac{cNR}{T} \cdot \frac{T}{c} + NR - \left[R\ln(T^c V/N) + S_0 - (c+1)R\right] N$$

$$= \frac{NR}{c} \cdot c + NR + NR\ln(T^c V/N) + NS_0 - (c+1)NR$$

整理すると、$T = U/(cNR)$を代入: $$S = NR\ln(T^c V/N) + NS_0$$

これは式(8)と一致する。

答え: $$S(U,V,N) = \frac{1}{T} U + \frac{p}{T} V - \frac{\mu}{T} N = NR\ln(T^c V/N) + NS_0$$

物理的意味:

  • エントロピーは広延変数（$U$, $V$, $N$）の一次同次関数である。
  • 各係数（$a_U$, $a_V$, $a_N$）は、対応する強度変数（$1/T$, $p/T$, $-\mu/T$）である。

セクションIV: 2成分理想気体混合

問1. 初期状態のエントロピー

問題: 体積$V$が仕切りで$V_1$と$V_2$に分けられ、それぞれに成分1（$N_1$モル）と成分2（$N_2$モル）が入っている平衡状態のエントロピーを、$(T, V, N_1, N_2)$の関数として表せ。

解答:

各成分のエントロピー: $$S_1(T, V_1, N_1) = N_1 R\ln(T^c V_1/N_1) + N_1 S_0$$ $$S_2(T, V_2, N_2) = N_2 R\ln(T^c V_2/N_2) + N_2 S_0$$

全エントロピー: $$S_{\text{tot}} = S_1 + S_2 = N_1 R\ln(T^c V_1/N_1) + N_2 R\ln(T^c V_2/N_2) + (N_1 + N_2) S_0$$

$V_1 + V_2 = V$であるが、平衡状態では圧力が等しい: $$p_1 = \frac{N_1 RT}{V_1} = p_2 = \frac{N_2 RT}{V_2} = p$$

したがって: $$\frac{N_1}{V_1} = \frac{N_2}{V_2} = \frac{N_1 + N_2}{V_1 + V_2} = \frac{N_1 + N_2}{V}$$

$$V_1 = \frac{N_1}{N_1 + N_2} V, \quad V_2 = \frac{N_2}{N_1 + N_2} V$$

したがって: $$S_{\text{tot}} = N_1 R\ln\left(\frac{T^c V_1}{N_1}\right) + N_2 R\ln\left(\frac{T^c V_2}{N_2}\right) + (N_1 + N_2) S_0$$

$$= N_1 R\ln\left(\frac{T^c V}{N_1 + N_2}\right) + N_2 R\ln\left(\frac{T^c V}{N_1 + N_2}\right) + (N_1 + N_2) S_0$$

$$= (N_1 + N_2) R\ln\left(\frac{T^c V}{N_1 + N_2}\right) + (N_1 + N_2) S_0$$

答え: $$S_{\text{tot}} = (N_1 + N_2) R\ln\left(\frac{T^c V}{N_1 + N_2}\right) + (N_1 + N_2) S_0$$

物理的意味:

  • 初期状態では、2つの気体は仕切りで分離されているが、圧力が平衡している。
  • エントロピーは、各成分が独立に占有する体積に依存する。

問2. 混合後のエントロピー

問題: 仕切りを取り除いて混合した後の平衡状態のエントロピー$S(T, V, N_1, N_2)$を計算せよ。

解答:

混合後の状態では、各成分が全体積$V$を占有する。理想気体では、各成分は独立に振る舞う（ダルトンの法則）。

成分1のエントロピー: $$S_1(T, V, N_1) = N_1 R\ln(T^c V/N_1) + N_1 S_0$$

成分2のエントロピー: $$S_2(T, V, N_2) = N_2 R\ln(T^c V/N_2) + N_2 S_0$$

全エントロピー: $$S(T, V, N_1, N_2) = S_1 + S_2 = N_1 R\ln(T^c V/N_1) + N_2 R\ln(T^c V/N_2) + (N_1 + N_2) S_0$$

答え: $$S(T, V, N_1, N_2) = N_1 R\ln\left(\frac{T^c V}{N_1}\right) + N_2 R\ln\left(\frac{T^c V}{N_2}\right) + (N_1 + N_2) S_0$$

物理的意味:

  • 混合後、各成分は全体積$V$を利用できるようになる。
  • エントロピーは、各成分がより大きな体積を占有できることにより増加する。

問3. エントロピー変化

問題: エントロピー変化$\Delta S = S(T, V, N_1, N_2) - {S_1(T, V_1, N_1) + S_2(T, V_2, N_2)}$を計算し、正負を判定せよ。

解答:

問1より: $$S_1(T, V_1, N_1) + S_2(T, V_2, N_2) = (N_1 + N_2) R\ln\left(\frac{T^c V}{N_1 + N_2}\right) + (N_1 + N_2) S_0$$

問2より: $$S(T, V, N_1, N_2) = N_1 R\ln\left(\frac{T^c V}{N_1}\right) + N_2 R\ln\left(\frac{T^c V}{N_2}\right) + (N_1 + N_2) S_0$$

したがって: $$\Delta S = N_1 R\ln\left(\frac{T^c V}{N_1}\right) + N_2 R\ln\left(\frac{T^c V}{N_2}\right) - (N_1 + N_2) R\ln\left(\frac{T^c V}{N_1 + N_2}\right)$$

$$= R\left[N_1\ln\left(\frac{V}{N_1}\right) + N_2\ln\left(\frac{V}{N_2}\right) - (N_1 + N_2)\ln\left(\frac{V}{N_1 + N_2}\right)\right]$$

$$= R\left[N_1\ln\left(\frac{N_1 + N_2}{N_1}\right) + N_2\ln\left(\frac{N_1 + N_2}{N_2}\right)\right]$$

$$= R\left[N_1\ln\left(1 + \frac{N_2}{N_1}\right) + N_2\ln\left(1 + \frac{N_1}{N_2}\right)\right]$$

対数の性質より、$\ln(1+x) > 0$（$x > 0$のとき）なので、$\Delta S > 0$。

答え: $$\Delta S = R\left[N_1\ln\left(1 + \frac{N_2}{N_1}\right) + N_2\ln\left(1 + \frac{N_1}{N_2}\right)\right] > 0$$

物理的意味:

  • エントロピーは混合により増加する（$\Delta S > 0$）。
  • これは、混合が自発的に起こることを示している（第二法則）。
  • 混合エントロピーは、各成分がより大きな体積を利用できることによる。

セクションV: ポアソン分布の導出

問1. 1個の分子が体積$v$に含まれる確率

問題: 体積$V$の容器に$N$個の分子がランダムに分布している。体積$v$の部分領域に1個の分子が含まれる確率$p$を求めよ。

解答:

分子の位置は完全にランダムであると仮定する。したがって、各分子が体積$v$に含まれる確率は、体積比に等しい:

$$p = \frac{v}{V}$$

答え: $$p = \frac{v}{V}$$

物理的意味:

  • 分子の位置がランダムであるため、特定の領域に存在する確率は、その領域の体積比に比例する。
  • これは、等確率の原理（等重率の原理）に基づいている。

問2. 平均分子数の予想

問題: 体積$v$に含まれる分子数$n$の平均値$\langle n \rangle$を予想せよ。

解答:

各分子が独立に確率$p$で体積$v$に含まれるため、期待値は:

$$\langle n \rangle = Np = N \cdot \frac{v}{V} = \frac{N}{V} \cdot v = \rho v$$

ここで、$\rho = N/V$は数密度である。

答え: $$\langle n \rangle = \rho v = \frac{Nv}{V}$$

物理的意味:

  • 平均分子数は、数密度と体積の積に等しい。
  • これは直感的に理解できる結果である。

問3. 確率分布の係数

問題: $n$個の分子が体積$v$に含まれる確率$P(n)$が $$P(n) = Ap^{\alpha}(1-p)^{\beta}$$ の形で与えられるとき、$A$, $\alpha$, $\beta$を求めよ。

解答:

$n$個の分子が体積$v$に含まれ、残りの$(N-n)$個が体積$v$の外にある確率は:

$$P(n) = \binom{N}{n} p^n (1-p)^{N-n}$$

ここで、$\binom{N}{n} = \frac{N!}{n!(N-n)!}$は二項係数である。

したがって: $$A = \binom{N}{n}, \quad \alpha = n, \quad \beta = N-n$$

答え: $$A = \binom{N}{n} = \frac{N!}{n!(N-n)!}, \quad \alpha = n, \quad \beta = N-n$$

物理的意味:

  • これは二項分布である。
  • $n$個の分子を選ぶ組み合わせの数が$\binom{N}{n}$通りあり、それぞれの確率が$p^n(1-p)^{N-n}$である。

問4. 母関数と統計量

問題: 母関数 $$F(x) = \sum_{n=0}^{N} x^n P(n)$$ を用いて、以下の関係式を示せ: $$\langle n \rangle = \left.\frac{dF(x)}{dx}\right|{x=1}$$ $$\langle n^2 \rangle - \langle n \rangle = \left.\frac{d^2F(x)}{dx^2}\right|{x=1}$$

解答:

ステップ1: 平均値の導出

$$F(x) = \sum_{n=0}^{N} x^n P(n)$$

微分: $$\frac{dF(x)}{dx} = \sum_{n=0}^{N} n x^{n-1} P(n)$$

$x = 1$を代入: $$\left.\frac{dF(x)}{dx}\right|{x=1} = \sum{n=0}^{N} n P(n) = \langle n \rangle$$

ステップ2: 分散の導出

2階微分: $$\frac{d^2F(x)}{dx^2} = \sum_{n=0}^{N} n(n-1) x^{n-2} P(n)$$

$x = 1$を代入: $$\left.\frac{d^2F(x)}{dx^2}\right|{x=1} = \sum{n=0}^{N} n(n-1) P(n) = \langle n(n-1) \rangle = \langle n^2 \rangle - \langle n \rangle$$

答え: 示された。

物理的意味:

  • 母関数は、確率分布のすべてのモーメント（平均、分散など）を計算するための便利なツールである。
  • 微分操作により、統計量を簡単に計算できる。

問5. 母関数の具体形と統計量の計算

問題: 問3の結果を母関数の定義に代入し、 $$F(x) = {(1-p) + px}^N$$ を示せ。また、これを用いて平均値$\langle n \rangle$と分散$\langle \delta n^2 \rangle$を計算せよ。

解答:

ステップ1: 母関数の計算

$$F(x) = \sum_{n=0}^{N} x^n P(n) = \sum_{n=0}^{N} x^n \binom{N}{n} p^n (1-p)^{N-n}$$

$$= \sum_{n=0}^{N} \binom{N}{n} (px)^n (1-p)^{N-n}$$

二項定理より: $$F(x) = {(1-p) + px}^N$$

ステップ2: 平均値の計算

$$\frac{dF(x)}{dx} = N {(1-p) + px}^{N-1} \cdot p$$

$x = 1$を代入: $$\langle n \rangle = N {(1-p) + p}^{N-1} \cdot p = N \cdot 1^{N-1} \cdot p = Np = \rho v$$

問2の予想と一致する。

ステップ3: 分散の計算

$$\frac{d^2F(x)}{dx^2} = N(N-1) {(1-p) + px}^{N-2} \cdot p^2$$

$x = 1$を代入: $$\langle n^2 \rangle - \langle n \rangle = N(N-1) p^2$$

したがって: $$\langle n^2 \rangle = N(N-1) p^2 + Np$$

分散: $$\langle \delta n^2 \rangle = \langle n^2 \rangle - \langle n \rangle^2 = N(N-1) p^2 + Np - (Np)^2$$

$$= N^2 p^2 - N p^2 + Np - N^2 p^2 = Np(1-p)$$

答え: $$F(x) = {(1-p) + px}^N$$ $$\langle n \rangle = Np = \rho v$$ $$\langle \delta n^2 \rangle = Np(1-p) = \rho v (1-p)$$

物理的意味:

  • 平均値は、数密度と体積の積に等しい。
  • 分散は、$Np(1-p)$で与えられる（二項分布の標準的な結果）。
  • $p \ll 1$のとき、分散は平均に近づく（ポアソン分布の特徴）。

問6. ポアソン分布への極限

問題: $\rho = N/V$を固定したまま、$v/V \to 0$の極限を取るとき、 $$P(n) = \frac{a^n}{n!} e^{-a}$$ （$a = \langle n \rangle$）となることを示せ。

解答:

ステップ1: 母関数の書き換え

$$F(x) = {(1-p) + px}^N = \exp[N\ln{(1-p) + px}]$$

ステップ2: 極限の計算

$p = v/V$であり、$v/V \to 0$の極限を考える。$\rho = N/V$を固定するので、$N \to \infty$、$p \to 0$、$Np = \rho v = a$（一定）。

$\ln{(1-p) + px}$を$p$について展開: $$\ln{(1-p) + px} = \ln{1 + p(x-1)} \approx p(x-1) - \frac{p^2(x-1)^2}{2} + \cdots$$

一次まで: $$\ln{(1-p) + px} \approx p(x-1)$$

したがって: $$F(x) \approx \exp[Np(x-1)] = \exp[a(x-1)] = e^{-a} e^{ax}$$

ステップ3: 確率分布の導出

$$F(x) = e^{-a} e^{ax} = e^{-a} \sum_{n=0}^{\infty} \frac{(ax)^n}{n!} = \sum_{n=0}^{\infty} \frac{a^n}{n!} e^{-a} x^n$$

一方、$F(x) = \sum_{n=0}^{\infty} x^n P(n)$なので、係数を比較:

$$P(n) = \frac{a^n}{n!} e^{-a}$$

答え: $$P(n) = \frac{a^n}{n!} e^{-a}$$

ここで、$a = \langle n \rangle = \rho v$である。

物理的意味:

  • ポアソン分布は、稀な事象（$p \ll 1$）の発生回数を記述する。
  • 平均値$a$のみで分布が決まる（分散も$a$に等しい）。
  • 分子数が非常に多く、観測領域が小さい場合、分子数の分布はポアソン分布に従う。
  • これは、統計力学における重要な分布であり、理想気体の粒子数揺らぎなどを記述する。

まとめ

本演習では、以下の重要な概念を学んだ:

  1. 熱力学関係式の導出: 偏微分とマクスウェル関係式を用いた熱力学量の関係式の導出
  2. 輪ゴムの熱力学: 温度依存性を持つバネ定数を持つ系の熱力学的記述
  3. 理想気体の統計力学: エントロピー、自由エネルギー、化学ポテンシャルの計算
  4. 混合エントロピー: 気体混合によるエントロピー増加
  5. ポアソン分布: 二項分布からポアソン分布への極限遷移

これらの概念は、統計力学の基礎を理解する上で重要である。
