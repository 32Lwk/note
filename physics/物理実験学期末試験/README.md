# 物理実験学期末試験 詳細解答

## ファイル構成

- **物理実験学期末試験_詳細解答.md**: 2024年度秋学期「物理実験学」期末試験の詳細解答
- **generate_physics_exam_figures.py**: 図を生成するPythonスクリプト
- **figures/**: 解答に使用される図ファイル（PNG形式）

## 使用方法

### 図の再生成

図を再生成する場合は、以下のコマンドを実行してください：

```bash
python3 generate_physics_exam_figures.py
```

### 必要なライブラリ

- matplotlib
- numpy
- scipy（ポアソン分布の計算に使用、オプション）

## 内容

本フォルダには、以下の大問の詳細解答が含まれています：

1. **大問1: グラフの描き方**
   - グラフの問題点の指摘
   - ログロググラフと片対数グラフの関数形
   - 適切なグラフの選択

2. **大問2: 誤差と有効数字**
   - 誤差伝播の公式
   - パイプの壁厚の計算
   - 有効数字の扱い

3. **大問3: 装置の読み取り**
   - ノギスによる内径測定
   - アナログテスターの読み取り
   - オシロスコープの波形読み取り

4. **大問4: 論理回路**
   - 論理回路の真理値表
   - 論理回路の名称
   - 4ビットカウンタの波形

5. **大問5: 確率と統計の基礎**
   - ガンマ線の検出頻度
   - ポアソン分布の適用
   - 分散の加法性

## 注意事項

- 図ファイルは `figures/` ディレクトリに保存されています
- Markdownファイル内の図へのパスは `figures/` となっています
- 図を再生成する場合は、`figures/` ディレクトリが存在することを確認してください
