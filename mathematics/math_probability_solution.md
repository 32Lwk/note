# 数学A 場合の数と確率 解答

## 問題5 【選択問題】 数学A 場合の数と確率（配点 50点）

---

## (1) 3つの袋からの球の取り出し

### 問題設定

3つの袋A, B, Cがあり、どの袋にも赤球が1個、白球が1個、黒球が1個の合計3個の球が入っている。A, B, Cからそれぞれ1個ずつ無作為に球を取り出す。

### (1-i) 取り出した3個の球がすべて赤球である確率

**解答:**

**ステップ1: 全事象の数**

各袋から1個ずつ球を取り出す場合の数は、各袋で3通りずつなので:
$$n(\Omega) = 3 \times 3 \times 3 = 27$$

**ステップ2: 事象「すべて赤球」の場合の数**

各袋から赤球を取り出す場合の数は、各袋で1通りずつなので:
$$n(A) = 1 \times 1 \times 1 = 1$$

**ステップ3: 確率の計算**

$$P(\text{すべて赤球}) = \frac{n(A)}{n(\Omega)} = \frac{1}{27}$$

**答え:** $\frac{1}{27}$

![問題(1)の確率](math_probability_fig1.png)

### (1-ii) 取り出した3個の球の色がすべて異なる確率

**解答:**

**ステップ1: 事象「すべて異なる色」の場合の数**

3個の球の色がすべて異なるためには、赤、白、黒が1個ずつ取り出される必要がある。

**方法1: 直接数える**

- 袋Aから赤球を取り出す: 1通り
- 袋Bから白球を取り出す: 1通り
- 袋Cから黒球を取り出す: 1通り

この組み合わせは1通り。しかし、どの袋からどの色を取り出すかは順列で考える必要がある。

**方法2: 順列で考える**

3つの袋から、それぞれ異なる色の球を取り出す。これは、3つの色（赤、白、黒）を3つの袋（A, B, C）に1つずつ割り当てる順列である。

$$n(B) = {}_3P_3 = 3! = 3 \times 2 \times 1 = 6$$

**ステップ2: 確率の計算**

$$P(\text{すべて異なる色}) = \frac{n(B)}{n(\Omega)} = \frac{6}{27} = \frac{2}{9}$$

**答え:** $\frac{2}{9}$

**補足説明:**

6通りの内訳:
1. A:赤, B:白, C:黒
2. A:赤, B:黒, C:白
3. A:白, B:赤, C:黒
4. A:白, B:黒, C:赤
5. A:黒, B:赤, C:白
6. A:黒, B:白, C:赤

### (1-iii) 取り出した3個の球のうち、ちょうど1個だけが赤球である確率

**解答:**

**ステップ1: 事象「ちょうど1個だけが赤球」の場合の数**

3個のうち、ちょうど1個だけが赤球である場合を考える。

**方法1: 直接数える**

赤球を取り出す袋を1つ選ぶ: ${}_3C_1 = 3$通り（A, B, Cのいずれか）

選んだ袋以外の2つの袋からは、白球または黒球を取り出す必要がある。

- 残り2つの袋から、白球と黒球を1個ずつ取り出す: $2 \times 1 = 2$通り
- 残り2つの袋から、白球を2個取り出す: $1 \times 1 = 1$通り
- 残り2つの袋から、黒球を2個取り出す: $1 \times 1 = 1$通り

したがって、赤球を取り出す袋を選んだ後、残り2つの袋からの取り出し方は $2 + 1 + 1 = 4$通り。

しかし、これは間違いである。正しくは:

**方法2: 正しい数え方**

赤球を取り出す袋を1つ選ぶ: ${}_3C_1 = 3$通り

残り2つの袋からは、赤球以外（白球または黒球）を取り出す。各袋で2通りずつなので:
$$2 \times 2 = 4\text{通り}$$

したがって:
$$n(C) = {}_3C_1 \times 2 \times 2 = 3 \times 4 = 12$$

**ステップ2: 確率の計算**

$$P(\text{ちょうど1個だけが赤球}) = \frac{n(C)}{n(\Omega)} = \frac{12}{27} = \frac{4}{9}$$

**答え:** $\frac{4}{9}$

**補足説明:**

12通りの内訳（赤球を取り出す袋ごと）:

**Aから赤球:**
- A:赤, B:白, C:白
- A:赤, B:白, C:黒
- A:赤, B:黒, C:白
- A:赤, B:黒, C:黒

**Bから赤球:**
- A:白, B:赤, C:白
- A:白, B:赤, C:黒
- A:黒, B:赤, C:白
- A:黒, B:赤, C:黒

**Cから赤球:**
- A:白, B:白, C:赤
- A:白, B:黒, C:赤
- A:黒, B:白, C:赤
- A:黒, B:黒, C:赤

## (2) 数直線上の点の移動と確率

### 問題設定

数直線上に2点P, Qがあり、それらの座標をそれぞれ$p$, $q$とする。P, Qは最初に原点にある。また、赤球2個、白球1個、黒球2個の合計5個の球が入った袋Fがある。

**操作:** Fから同時に3個の球を取り出し、取り出された球に含まれる赤球の個数と同じ長さだけPを正の方向に動かし、白球が含まれていればQを正の方向に1動かす。その後、取り出された球を袋Fに戻す。この操作を何回か繰り返す。

![操作の流れの可視化](math_probability_fig5.png)

### (2-i) 1回の操作後、$p=1$, $q=1$となる確率

**解答:**

**ステップ1: 条件の整理**

$p=1$, $q=1$となるためには:
- 取り出された球に含まれる赤球の個数が1個（Pが1動く）
- 白球が含まれている（Qが1動く）

**ステップ2: 全事象の数**

袋Fから3個の球を取り出す場合の数は、5個から3個を選ぶ組み合わせ:
$$n(\Omega) = {}_5C_3 = \frac{5!}{3!(5-3)!} = \frac{5 \times 4}{2 \times 1} = 10$$

**ステップ3: 事象「赤球1個、白球含む」の場合の数**

取り出す3個の球が「赤球1個、白球1個、黒球1個」である場合を考える。

- 赤球2個から1個選ぶ: ${}_2C_1 = 2$通り
- 白球1個から1個選ぶ: ${}_1C_1 = 1$通り
- 黒球2個から1個選ぶ: ${}_2C_1 = 2$通り

したがって:
$$n(D) = {}_2C_1 \times {}_1C_1 \times {}_2C_1 = 2 \times 1 \times 2 = 4$$

**ステップ4: 確率の計算**

$$P(p=1, q=1) = \frac{n(D)}{n(\Omega)} = \frac{4}{10} = \frac{2}{5}$$

**答え:** $\frac{2}{5}$

![問題(2-i)の確率分布](math_probability_fig2.png)

**補足説明:**

4通りの内訳:
1. 赤球①, 白球, 黒球①
2. 赤球①, 白球, 黒球②
3. 赤球②, 白球, 黒球①
4. 赤球②, 白球, 黒球②

---

### (2-ii) 2回の操作後、$p=3$, $q=1$となる確率

**解答:**

**ステップ1: 条件の整理**

2回の操作後、$p=3$, $q=1$となるためには:
- 2回の操作で、Pの移動距離の合計が3（$p=3$）
- 2回の操作で、Qが1回だけ動く（$q=1$）

**ステップ2: Pの移動距離の組み合わせ**

Pの移動距離は、取り出された赤球の個数に等しい。2回の操作で合計3になる組み合わせは:
- 1回目: 赤球1個（移動距離1）、2回目: 赤球2個（移動距離2）→ 合計3
- 1回目: 赤球2個（移動距離2）、2回目: 赤球1個（移動距離1）→ 合計3
- 1回目: 赤球0個（移動距離0）、2回目: 赤球3個（移動距離3）→ 合計3
- 1回目: 赤球3個（移動距離3）、2回目: 赤球0個（移動距離0）→ 合計3

しかし、袋Fには赤球が2個しかないため、1回の操作で赤球3個を取り出すことは不可能である。

したがって、可能な組み合わせは:
- 1回目: 赤球1個、2回目: 赤球2個
- 1回目: 赤球2個、2回目: 赤球1個

**ステップ3: Qの移動条件**

$q=1$となるためには、2回の操作のうち、ちょうど1回だけ白球が含まれる必要がある。

**ケース1: 1回目で赤球1個、2回目で赤球2個**

- 1回目: 赤球1個、白球含む → P: +1, Q: +1
- 2回目: 赤球2個、白球含まない → P: +2, Q: +0

この場合、$p=3$, $q=1$となる。

- 1回目: 赤球1個、白球含まない → P: +1, Q: +0
- 2回目: 赤球2個、白球含む → P: +2, Q: +1

この場合も、$p=3$, $q=1$となる。

**ケース2: 1回目で赤球2個、2回目で赤球1個**

- 1回目: 赤球2個、白球含む → P: +2, Q: +1
- 2回目: 赤球1個、白球含まない → P: +1, Q: +0

この場合、$p=3$, $q=1$となる。

- 1回目: 赤球2個、白球含まない → P: +2, Q: +0
- 2回目: 赤球1個、白球含む → P: +1, Q: +1

この場合も、$p=3$, $q=1$となる。

**ステップ4: 各ケースの確率の計算**

**ケース1-1: 1回目で赤球1個・白球含む、2回目で赤球2個・白球含まない**

1回目の確率:
- 赤球1個、白球1個、黒球1個を取り出す確率: $\frac{4}{10} = \frac{2}{5}$（(2-i)より）

2回目の確率:
- 赤球2個、黒球1個を取り出す確率を計算する
- 赤球2個から2個選ぶ: ${}_2C_2 = 1$通り
- 黒球2個から1個選ぶ: ${}_2C_1 = 2$通り
- 白球は含まないので、白球0個: ${}_1C_0 = 1$通り
- 場合の数: $1 \times 2 \times 1 = 2$
- 確率: $\frac{2}{10} = \frac{1}{5}$

したがって:
$$P_1 = \frac{2}{5} \times \frac{1}{5} = \frac{2}{25}$$

**ケース1-2: 1回目で赤球1個・白球含まない、2回目で赤球2個・白球含む**

1回目の確率:
- 赤球1個、黒球2個を取り出す確率を計算する
- 赤球2個から1個選ぶ: ${}_2C_1 = 2$通り
- 黒球2個から2個選ぶ: ${}_2C_2 = 1$通り
- 白球は含まないので、白球0個: ${}_1C_0 = 1$通り
- 場合の数: $2 \times 1 \times 1 = 2$
- 確率: $\frac{2}{10} = \frac{1}{5}$

2回目の確率:
- 赤球2個、白球1個を取り出す確率を計算する
- 赤球2個から2個選ぶ: ${}_2C_2 = 1$通り
- 白球1個から1個選ぶ: ${}_1C_1 = 1$通り
- 黒球は含まれないので、黒球0個: ${}_2C_0 = 1$通り
- 場合の数: $1 \times 1 \times 1 = 1$
- 確率: $\frac{1}{10}$

したがって:
$$P_2 = \frac{1}{5} \times \frac{1}{10} = \frac{1}{50}$$

**ケース2-1: 1回目で赤球2個・白球含む、2回目で赤球1個・白球含まない**

1回目の確率:
- 赤球2個、白球1個を取り出す確率: $\frac{1}{10}$（ケース1-2の2回目と同じ）

2回目の確率:
- 赤球1個、黒球2個を取り出す確率: $\frac{1}{5}$（ケース1-2の1回目と同じ）

したがって:
$$P_3 = \frac{1}{10} \times \frac{1}{5} = \frac{1}{50}$$

**ケース2-2: 1回目で赤球2個・白球含まない、2回目で赤球1個・白球含む**

1回目の確率:
- 赤球2個、黒球1個を取り出す確率: $\frac{1}{5}$（ケース1-1の2回目と同じ）

2回目の確率:
- 赤球1個、白球1個、黒球1個を取り出す確率: $\frac{2}{5}$（(2-i)と同じ）

したがって:
$$P_4 = \frac{1}{5} \times \frac{2}{5} = \frac{2}{25}$$

**ステップ5: 全確率の計算**

$$P(p=3, q=1) = P_1 + P_2 + P_3 + P_4 = \frac{2}{25} + \frac{1}{50} + \frac{1}{50} + \frac{2}{25}$$

$$= \frac{4}{50} + \frac{1}{50} + \frac{1}{50} + \frac{4}{50} = \frac{10}{50} = \frac{1}{5}$$

**答え:** $\frac{1}{5}$

![問題(2-ii)の各ケースの確率](math_probability_fig3.png)

**補足説明:**

各ケースの詳細:

| ケース | 1回目 | 2回目 | 確率 |
|--------|-------|-------|------|
| 1-1 | 赤1・白1・黒1 | 赤2・黒1 | $\frac{2}{5} \times \frac{1}{5} = \frac{2}{25}$ |
| 1-2 | 赤1・黒2 | 赤2・白1 | $\frac{1}{5} \times \frac{1}{10} = \frac{1}{50}$ |
| 2-1 | 赤2・白1 | 赤1・黒2 | $\frac{1}{10} \times \frac{1}{5} = \frac{1}{50}$ |
| 2-2 | 赤2・黒1 | 赤1・白1・黒1 | $\frac{1}{5} \times \frac{2}{5} = \frac{2}{25}$ |

合計: $\frac{2}{25} + \frac{1}{50} + \frac{1}{50} + \frac{2}{25} = \frac{4 + 1 + 1 + 4}{50} = \frac{10}{50} = \frac{1}{5}$

### (2-iii) 4回の操作後に$p+q=4$となったとき、$p>q$である条件付き確率

**解答:**

**ステップ1: 条件付き確率の定義**

$$P(p>q \mid p+q=4) = \frac{P(p>q \cap p+q=4)}{P(p+q=4)}$$

**ステップ2: $p+q=4$となる場合の列挙**

4回の操作で$p+q=4$となる組み合わせを考える。

$p$は赤球の個数の合計、$q$は白球が含まれた回数である。

可能な組み合わせ:
- $(p, q) = (4, 0)$: 4回とも白球を含まず、赤球の合計が4
- $(p, q) = (3, 1)$: 1回だけ白球を含み、赤球の合計が3
- $(p, q) = (2, 2)$: 2回白球を含み、赤球の合計が2
- $(p, q) = (1, 3)$: 3回白球を含み、赤球の合計が1
- $(p, q) = (0, 4)$: 4回とも白球を含み、赤球の合計が0

**ステップ3: 各組み合わせの確率の計算**

各操作は独立なので、4回の操作の確率は各回の確率の積である。

**準備: 1回の操作で取り出される球の組み合わせと確率**

袋Fから3個の球を取り出す場合の数は10通り（(2-i)より）。

各組み合わせを列挙:

1. 赤2・白1・黒0: ${}_2C_2 \times {}_1C_1 \times {}_2C_0 = 1 \times 1 \times 1 = 1$通り → 確率$\frac{1}{10}$
2. 赤2・白0・黒1: ${}_2C_2 \times {}_1C_0 \times {}_2C_1 = 1 \times 1 \times 2 = 2$通り → 確率$\frac{2}{10} = \frac{1}{5}$
3. 赤1・白1・黒1: ${}_2C_1 \times {}_1C_1 \times {}_2C_1 = 2 \times 1 \times 2 = 4$通り → 確率$\frac{4}{10} = \frac{2}{5}$
4. 赤1・白0・黒2: ${}_2C_1 \times {}_1C_0 \times {}_2C_2 = 2 \times 1 \times 1 = 2$通り → 確率$\frac{2}{10} = \frac{1}{5}$
5. 赤0・白1・黒2: ${}_2C_0 \times {}_1C_1 \times {}_2C_2 = 1 \times 1 \times 1 = 1$通り → 確率$\frac{1}{10}$

合計: $1 + 2 + 4 + 2 + 1 = 10$通り ✓

各組み合わせでのP, Qの移動:
- 1. 赤2・白1: P +2, Q +1
- 2. 赤2・黒1: P +2, Q +0
- 3. 赤1・白1・黒1: P +1, Q +1
- 4. 赤1・黒2: P +1, Q +0
- 5. 赤0・白1・黒2: P +0, Q +1

**ケースA: $(p, q) = (4, 0)$**

4回とも白球を含まず、赤球の合計が4。

可能な組み合わせ:
- 2回目: 赤2・黒1（確率$\frac{1}{5}$）、2回目: 赤2・黒1（確率$\frac{1}{5}$）
- その他の組み合わせも可能

より系統的に考えると、4回の操作で:
- 赤球の合計が4: 赤2が2回、または赤1が4回、または赤2が1回と赤1が2回など
- 白球を含まない: すべての操作で白球を含まない

可能なパターン:
1. 赤2・黒1が2回: 確率$\left(\frac{1}{5}\right)^2 = \frac{1}{25}$
2. 赤1・黒2が4回: 確率$\left(\frac{1}{5}\right)^4 = \frac{1}{625}$

しかし、赤球の合計が4になるためには、赤2が2回取り出される必要がある（赤球は2個しかないため、1回の操作で最大2個まで）。

したがって、唯一の可能性は:
- 赤2・黒1が2回: 確率$\left(\frac{1}{5}\right)^2 = \frac{1}{25}$

しかし、4回の操作のうち、どの2回で赤2・黒1を取り出すかは${}_4C_2 = 6$通りある。残り2回は、赤球を含まない組み合わせ（赤0・白1・黒2）を取り出す必要があるが、これは白球を含むため条件に合わない。

再考: 赤球の合計が4で、白球を含まない場合。

1回の操作で取り出せる赤球の個数は0, 1, 2のいずれか。4回の操作で合計4になる組み合わせ:
- 赤2が2回、赤0が2回: 赤2・黒1が2回、赤0・白1・黒2が2回 → しかし、これは白球を含む
- 赤2が2回、赤1が0回、赤0が2回: 不可能（白球を含まない組み合わせで赤0は存在しない）

実際、白球を含まない組み合わせは:
- 赤2・黒1: 赤2個
- 赤1・黒2: 赤1個

したがって、$(p, q) = (4, 0)$は不可能である。

**ケースB: $(p, q) = (3, 1)$**

1回だけ白球を含み、赤球の合計が3。

可能なパターン:
- 赤2・白1が1回、赤1・黒2が2回、赤0・白1・黒2が1回 → しかし、これは白球を含む回数が2回
- 赤2・白1が1回、赤1・黒2が1回、赤2・黒1が1回、赤0・白1・黒2が1回 → 白球を含む回数が2回

より系統的に:

白球を含む操作が1回、白球を含まない操作が3回。

白球を含む操作（1回）:
- 赤2・白1: P +2, Q +1（確率$\frac{1}{10}$）
- 赤1・白1・黒1: P +1, Q +1（確率$\frac{2}{5}$）
- 赤0・白1・黒2: P +0, Q +1（確率$\frac{1}{10}$）

白球を含まない操作（3回）:
- 赤2・黒1: P +2, Q +0（確率$\frac{1}{5}$）
- 赤1・黒2: P +1, Q +0（確率$\frac{1}{5}$）

$p=3$となるためには、白球を含む操作と白球を含まない操作の赤球の合計が3。

可能な組み合わせ:
1. 白球を含む: 赤2・白1（P +2）、白球を含まない: 赤1・黒2が1回（P +1）、赤2・黒1が0回、赤1・黒2が2回は不可能（赤球の合計が3にならない）
2. 白球を含む: 赤1・白1・黒1（P +1）、白球を含まない: 赤2・黒1が1回（P +2）
3. 白球を含む: 赤0・白1・黒2（P +0）、白球を含まない: 赤2・黒1が1回（P +2）、赤1・黒2が1回（P +1）

しかし、白球を含まない操作が3回なので、より詳細に考える必要がある。

**より系統的なアプローチ:**

4回の操作で、各操作の結果を$(p_i, q_i)$と表す（$i = 1, 2, 3, 4$）。

可能な$(p_i, q_i)$の値:
- $(2, 1)$: 確率$\frac{1}{10}$（赤2・白1）
- $(2, 0)$: 確率$\frac{1}{5}$（赤2・黒1）
- $(1, 1)$: 確率$\frac{2}{5}$（赤1・白1・黒1）
- $(1, 0)$: 確率$\frac{1}{5}$（赤1・黒2）
- $(0, 1)$: 確率$\frac{1}{10}$（赤0・白1・黒2）

$p+q=4$となる組み合わせを列挙:

1. $(2,1), (2,0), (0,0), (0,0)$: $p=4, q=1$ → $p+q=5$ ✗
2. $(2,1), (1,0), (1,0), (0,0)$: $p=4, q=1$ → $p+q=5$ ✗
3. $(2,1), (1,0), (0,1), (0,0)$: $p=3, q=2$ → $p+q=5$ ✗
4. $(2,0), (2,0), (0,1), (0,1)$: $p=4, q=2$ → $p+q=6$ ✗

実際、$(0,0)$は存在しない（どの操作でも必ず何らかの球を取り出す）。

正しく列挙し直す:

$p+q=4$となる組み合わせ:
- $(p, q) = (4, 0)$: 不可能（上記の理由）
- $(p, q) = (3, 1)$: 可能
- $(p, q) = (2, 2)$: 可能
- $(p, q) = (1, 3)$: 可能
- $(p, q) = (0, 4)$: 可能

**$(p, q) = (3, 1)$の場合:**

白球を含む操作が1回、白球を含まない操作が3回で、$p=3$。

可能な組み合わせ:
1. 白球を含む: $(2,1)$、白球を含まない: $(1,0)$が1回、$(0,0)$が2回 → 不可能
2. 白球を含む: $(1,1)$、白球を含まない: $(2,0)$が1回、$(0,0)$が2回 → 不可能
3. 白球を含む: $(0,1)$、白球を含まない: $(2,0)$が1回、$(1,0)$が1回、$(0,0)$が1回 → 不可能

実際、白球を含まない操作で取り出せるのは$(2,0)$または$(1,0)$のみ。

$p=3$で$q=1$となる組み合わせ:
- 白球を含む: $(1,1)$、白球を含まない: $(2,0)$が1回 → しかし、操作は4回なので、残り2回は何か取り出す必要がある

4回の操作の合計を考えると、各操作は独立なので、組み合わせを正確に数える必要がある。

**より簡潔なアプローチ:**

各操作の結果を$X_i = (p_i, q_i)$と表す。

$p+q=4$となる条件は、$\sum_{i=1}^4 p_i + \sum_{i=1}^4 q_i = 4$。

各$X_i$の可能な値と確率:
- $X = (2, 1)$: $P = \frac{1}{10}$
- $X = (2, 0)$: $P = \frac{1}{5}$
- $X = (1, 1)$: $P = \frac{2}{5}$
- $X = (1, 0)$: $P = \frac{1}{5}$
- $X = (0, 1)$: $P = \frac{1}{10}$

$p+q=4$となる組み合わせを、各操作の$p_i + q_i$の合計が4になるように選ぶ。

各操作の$p_i + q_i$の値:
- $(2, 1)$: $p_i + q_i = 3$
- $(2, 0)$: $p_i + q_i = 2$
- $(1, 1)$: $p_i + q_i = 2$
- $(1, 0)$: $p_i + q_i = 1$
- $(0, 1)$: $p_i + q_i = 1$

4回の操作で合計が4になる組み合わせ:
- $1 + 1 + 1 + 1 = 4$: $(1,0)$が4回、または$(0,1)$が4回、または$(1,0)$と$(0,1)$の組み合わせ
- $2 + 1 + 1 + 0 = 4$: 不可能（$0$は存在しない）
- $2 + 2 + 0 + 0 = 4$: 不可能
- $3 + 1 + 0 + 0 = 4$: 不可能
- $2 + 1 + 1 = 4$（3回）: 不可能（操作は4回）

実際、各操作の$p_i + q_i$の最小値は1（$(1,0)$または$(0,1)$）なので、4回の操作で最小値は4。したがって、$p+q=4$となるためには、すべての操作で$p_i + q_i = 1$である必要がある。

すなわち:
- $(1, 0)$が4回: $p=4, q=0$ → $p+q=4$ ✓
- $(0, 1)$が4回: $p=0, q=4$ → $p+q=4$ ✓
- $(1, 0)$が$k$回、$(0, 1)$が$(4-k)$回: $p=k, q=4-k$ → $p+q=4$ ✓

したがって、$p+q=4$となる組み合わせは:
- $(p, q) = (4, 0)$: $(1,0)$が4回
- $(p, q) = (3, 1)$: $(1,0)$が3回、$(0,1)$が1回
- $(p, q) = (2, 2)$: $(1,0)$が2回、$(0,1)$が2回
- $(p, q) = (1, 3)$: $(1,0)$が1回、$(0,1)$が3回
- $(p, q) = (0, 4)$: $(0,1)$が4回

**各ケースの確率:**

**ケース1: $(p, q) = (4, 0)$**

$(1,0)$が4回: 確率$\left(\frac{1}{5}\right)^4 = \frac{1}{625}$

**ケース2: $(p, q) = (3, 1)$**

$(1,0)$が3回、$(0,1)$が1回。

どの回で$(0,1)$を取り出すか: ${}_4C_1 = 4$通り

確率: ${}_4C_1 \times \left(\frac{1}{5}\right)^3 \times \frac{1}{10} = 4 \times \frac{1}{125} \times \frac{1}{10} = \frac{4}{1250} = \frac{2}{625}$

**ケース3: $(p, q) = (2, 2)$**

$(1,0)$が2回、$(0,1)$が2回。

どの2回で$(0,1)$を取り出すか: ${}_4C_2 = 6$通り

確率: ${}_4C_2 \times \left(\frac{1}{5}\right)^2 \times \left(\frac{1}{10}\right)^2 = 6 \times \frac{1}{25} \times \frac{1}{100} = \frac{6}{2500} = \frac{3}{1250}$

**ケース4: $(p, q) = (1, 3)$**

$(1,0)$が1回、$(0,1)$が3回。

どの1回で$(1,0)$を取り出すか: ${}_4C_1 = 4$通り

確率: ${}_4C_1 \times \frac{1}{5} \times \left(\frac{1}{10}\right)^3 = 4 \times \frac{1}{5} \times \frac{1}{1000} = \frac{4}{5000} = \frac{2}{2500} = \frac{1}{1250}$

**ケース5: $(p, q) = (0, 4)$**

$(0,1)$が4回: 確率$\left(\frac{1}{10}\right)^4 = \frac{1}{10000}$

**ステップ4: $P(p+q=4)$の計算**

$$P(p+q=4) = \frac{1}{625} + \frac{2}{625} + \frac{3}{1250} + \frac{1}{1250} + \frac{1}{10000}$$

通分:
- $\frac{1}{625} = \frac{16}{10000}$
- $\frac{2}{625} = \frac{32}{10000}$
- $\frac{3}{1250} = \frac{24}{10000}$
- $\frac{1}{1250} = \frac{8}{10000}$
- $\frac{1}{10000} = \frac{1}{10000}$

$$P(p+q=4) = \frac{16 + 32 + 24 + 8 + 1}{10000} = \frac{81}{10000}$$

**ステップ5: $P(p>q \cap p+q=4)$の計算**

$p>q$かつ$p+q=4$となる組み合わせ:
- $(p, q) = (4, 0)$: $p=4 > 0 = q$ ✓
- $(p, q) = (3, 1)$: $p=3 > 1 = q$ ✓
- $(p, q) = (2, 2)$: $p=2 = 2 = q$ ✗
- $(p, q) = (1, 3)$: $p=1 < 3 = q$ ✗
- $(p, q) = (0, 4)$: $p=0 < 4 = q$ ✗

したがって:
$$P(p>q \cap p+q=4) = \frac{1}{625} + \frac{2}{625} = \frac{3}{625} = \frac{48}{10000}$$

**ステップ6: 条件付き確率の計算**

$$P(p>q \mid p+q=4) = \frac{P(p>q \cap p+q=4)}{P(p+q=4)} = \frac{\frac{48}{10000}}{\frac{81}{10000}} = \frac{48}{81} = \frac{16}{27}$$

**答え:** $\frac{16}{27}$

**補足説明:**

各ケースの詳細:

| $(p, q)$ | 組み合わせ | 確率 | $p>q$? |
|----------|-----------|------|--------|
| $(4, 0)$ | $(1,0)$が4回 | $\frac{1}{625} = \frac{16}{10000}$ | ✓ |
| $(3, 1)$ | $(1,0)$が3回、$(0,1)$が1回 | $\frac{2}{625} = \frac{32}{10000}$ | ✓ |
| $(2, 2)$ | $(1,0)$が2回、$(0,1)$が2回 | $\frac{3}{1250} = \frac{24}{10000}$ | ✗ |
| $(1, 3)$ | $(1,0)$が1回、$(0,1)$が3回 | $\frac{1}{1250} = \frac{8}{10000}$ | ✗ |
| $(0, 4)$ | $(0,1)$が4回 | $\frac{1}{10000}$ | ✗ |

$p>q$となる確率: $\frac{16 + 32}{10000} = \frac{48}{10000}$

全確率: $\frac{81}{10000}$

条件付き確率: $\frac{48}{81} = \frac{16}{27}$

## まとめ

本問題では、以下の確率の概念を扱った:

1. **基本的な確率:** 全事象に対する事象の割合
2. **独立試行:** 複数回の操作が独立である場合の確率
3. **条件付き確率:** ある条件の下での確率

特に(2-iii)では、4回の操作の結果を系統的に列挙し、条件付き確率を計算した。
